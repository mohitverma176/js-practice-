# 🚀 JavaScript Basics Projects

This repository contains beginner-friendly projects to learn and practice core concepts of JavaScript. Each project helps in building a strong foundation in web development.

---

## 📖 About

This project includes simple JavaScript applications to understand:

- Variables & Data Types  
- Functions  
- Loops & Conditions  
- DOM Manipulation  
- Event Handling
